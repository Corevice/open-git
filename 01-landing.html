5分でデプロイ完了

DockerでもKubernetesでも、お好みの方法で本番運用を開始できます。

🐳

Docker Compose

シングルノードでの検証・小規模運用に最適。公式イメージを使って数分で起動できます。

# compose.yamlをダウンロード
$ curl -L openforge.io/install > compose.yaml
# 起動
$ docker compose up -d
$ open http://localhost:3000
☸️

Kubernetes (Helm)

本番グレードのHA構成、自動スケーリング、ロールバック対応。公式Helm Chartで簡単導入。

# Helmリポジトリを追加
$ helm repo add openforge \
    https://charts.openforge.io
# インストール
$ helm install forge openforge/openforge
