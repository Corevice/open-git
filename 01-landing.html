5分でデプロイ。

DockerでもKubernetes (Helm) でも、お好みの方法でセルフホスト。

🐳 Docker Compose docker-compose.yml 
# 単一コマンドで起動
version: '3.8'
services:
  openforge:
    image: openforge/server:latest
    ports:
      - "3000:3000"
      - "2222:22"
    volumes:
      - ./data:/var/lib/openforge
    environment:
      OPENFORGE_DOMAIN: git.example.com
☸️ Helm Chart install.sh 
# Helmリポジトリを追加
$ helm repo add openforge \
    https://charts.openforge.io

# Kubernetesクラスタにインストール
$ helm install my-forge openforge/openforge \
    --namespace forge \
    --create-namespace \
    --set domain=git.example.com \
    --set persistence.size=100Gi

# 完了！
